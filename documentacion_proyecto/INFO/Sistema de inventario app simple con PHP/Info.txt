SISTEMA DE INVENTARIO APP SIMPLE CON PHP

 YESID SANCHEZ RAMOS  16:14:00

El sistema de Inventario App Simple es un aplicación web de inventario,   que le permitirá mantener el seguimiento de la cantidad y el precio de los artículos existentes en su almacén o negocio. Cuando se crea un nuevo producto, se puede establecer una cantidad, y entonces todo es controlado por el registro de entrada/salida de productos al inventario, el sistema permite registrar cada movimiento que se realiza a través del sistema.

Características principales:

Estadísticas sobre su almacén

4 funciones de usuario diferentes (Administrador, Supervisor General, supervisor y vendedor)

Diseño de Responsivo

Artículos agrupados por categorías

Tener seguimiento del lugar donde se encuentran las categorías

Registros de cada movimiento realizado en el sistema

Agregación, edición y eliminación de producto a la base de datos.

Agregación, edición y eliminación de categorías a la base de datos.

Manejo de perfiles de usuarios

Modulo de configuración del sistema.


Instalación en windows (servidor local)

Descargar los archivos fuentes del sistema

Copiar y descomprimir el archivo en la carpeta c:\xampp\htdocs, al final tendrás una carpeta llamada “inventario_app_simple”, a la cual podrás acceder desde el navegador como: http://localhost/inventario_app_simple/

Crear una base de datos usando PHPMyAdmin accediendo a la url siguiente: http://localhost/phpmyadmin/

Importar las tablas de la base de datos para ello vamos a buscar el archivo “invent_db.sql” , el cual se encuentra en la carpeta llamada “db” de nuestro sistema, una vez localizado procedemos a hacer la importación de los datos desde PHPMyAdmin

Configurar los datos de conexión a la base de datos editando el archivo de configuración: “config.php“,  que se encuentra en el directorio root del sistema.

Vista web: http://localhost/inventario_app_simple/
